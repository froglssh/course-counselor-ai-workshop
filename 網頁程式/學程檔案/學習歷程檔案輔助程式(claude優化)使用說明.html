🎓 學習歷程檔案輔助程式 2.0 使用說明與指南

輔助高中生高效撰寫符合 108 課綱與升學指南規格的備審資料、多元表現與學習歷程自述。

🔗 開啟線上版程式

👤 1. 基礎設定與基本資料

在使用程式填寫任何內容前，建議先在左側邊欄的 **👤 基本資料** 中填入以下欄位，這將會大幅優化 AI 自動寫作的客製化效果：

👤 學生姓名

用以生成個人化的成果報告與 Word/PDF 檔中的署名資訊。

🎯 目標校系

如「台大資工系」、「師大生科系」。AI 優化時會主動以此校系特質做連結，調整論述深度。

⚙️ 2. 主控制區功能操作

頁首右上角的四大功能按鈕能有效管理您的所有寫作資料：

⚙️ AI 設定

設定 OpenAI/Gemini 的 API 金鑰與模型偏好。您可以輸入個人 API Key 進行直接串接，或者自訂 AI 提示詞架構，使 AI 輸出的風格最符合您的調性。

💾 儲存

一鍵將目前**所有分頁的文字資料與佐證圖片欄位**儲存至您的瀏覽器本地暫存（LocalStorage）。即使不小心關閉瀏覽器，資料也不會遺失。

📂 讀取

從本機瀏覽器中還原上一次儲存的所有輸入狀態，方便隨時繼續編輯。

🗑️ 刪除

完全清空目前輸入的所有內容，並清除本地瀏覽器的暫存紀錄，重新開始撰寫新檔案。

📁 3. 十大學習歷程檔案類型詳解

本程式根據四份升學指南 PDF 精華，為十種學習歷程主要檔案類型設計了專用的引導欄位與撰寫原則：

📘 課程學習成果 (Course)

修課產出的作品（B 項）。學術能力的直接證據，重質不重量。

  • 標題與主題：交代作品名稱與所屬課程。
  • 100字簡述：教授的預告片！使用「What＋How＋Result」黃金公式。
  • 摘要、作品說明、正文、心得反思：交代製作動機與核心能力成長。
公式：以「大巨蛋對周邊房價影響」為題，透過實價登錄數據分析，發現正相關但有滯後效應，培養數據分析能力。

🌟 多元表現 (Diverse)

校內外非修課成果（C 項）：社團、幹部、競賽、檢定、志工等。

  • 項目名稱與類型、擔任角色、投入歷程、具體成果、科系連結。
  • 重點在於：戴上「科系濾鏡」，強調你「解決了什麼」「帶領了什麼」，用數據佐證。

🧩 多元表現綜整心得 (Diverse Summary)

獨立上傳的 PDF（最多 800 字＋3 張圖片），為三年課外精華的去蕪存菁。

  • 個人標籤、選材清單、三分段落、結論與圖片規劃。
  • 設定一個「個人標籤」（例如：數據驅動的探究者）串聯不同活動，避免流水帳。

📖 學習歷程自述 (Self Statement)

把三年零散片段串成完整故事的「自我行銷提案書」，採 N-O-P-Q 架構。

  • O.高中反思（STAR 原則描述關鍵學習事件）、P.就讀動機、校系特色打擊與Q.未來計畫。
  • 前後必須緊密扣合，展現對該科系課程、實驗室的深入研究。

🔍 自主學習計畫與成果 (Self Directed)

展現「自訂主題 → 找資源 → 執行 → 反思」的自主學習能力。

  • 主題動機、可檢核目標、方法時程、執行歷程、成果與反思。
  • 重點在於誠實記錄調整與卡關，展現科學韌性與解決問題的態度。

🧪 探究與實作成果 (Inquiry)

大學教授最看重的項目之一，採用 IMRD（引言-方法-結果-討論）論證結構。

  • 研究問題假設、動機背景、方法、結果數據、分析討論、結論與延伸探究。
  • 自然科重「論證與建模」（需含誤差分析）；社會科重「多重觀點與社會關懷」。

🏆 競賽成果 (Competition)

科展、學科競賽或專題競賽成果，展現抗壓力與問題解決力。

  • 競賽名稱、背景難度、你的角色分工、準備歷程、量化成果、習得能力、科系連結。
  • 重點不在於得名與否，而是遭遇瓶頸時的具體突破過程。

✍️ 個人自傳 (Autobiography)

個人品牌白皮書。是「價值提案」而非單純的自我揭露（建議 800–1200 字）。

  • 黃金開場、成長背景、求學歷程、社團活動、經歷亮點、未來展望。
  • 開場請用 3–5 句「自我摘要」破題，並善用強動詞與數據呈現重要成就。

🗺️ 讀書計畫 (Study Plan)

驗證就讀動機可行性的試金石，展示大學四年的執行藍圖。

  • 近程（轉銜學習）、中程大一大二（奠基）、中程大三大四（深化）、遠程（生涯願景）。
  • 切忌複製系網課表或流於大方向，要具體列出期望修讀的特殊課名、研究專題與目標實驗室。

🚀 未來發展 (Future)

展現清晰的生涯規劃與成長心態，連結所學與社會貢獻。

  • 大學研究所規劃、畢業後十年規劃、終生志業、科系連結、社會回饋。
  • 讓教授看到你的遠見，以及你如何將所學回饋社會，提升思考格局。

✨ 4. 欄位寫作與 AI 優化工具

在每個輸入欄位的右下角，皆設有三個便捷的輔助按鈕：

🎤 語音輸入

呼叫瀏覽器語音辨識 API，用說的也能快速打字，最適合靈感爆發時的隨筆記錄。

✨ AI 優化此欄

針對該特定輸入欄位進行內容升級，共支援三種靈活的工作模式：

1. 本地系統優化 離線・免金鑰

內建演算法自動修復常見錯別字、贅字清理、字數檢查，並自動提供台灣升學術語的校正建議。

2. 產生提示詞並複製 手動貼上

一鍵生成最精準的優化 Prompt 指令，包裝你的原文，直接複製後開啟網頁版 ChatGPT 或 Gemini 進行問答。

3. 直接呼叫 API 即時完成

直接在程式視窗內呼叫 AI API 進行流暢潤飾，滿意後即可一鍵套用回原欄位。

✕ 清除

快速清空該欄位的文字內容，方便重新填寫。

📎 5. 佐證資料與成果匯出

當您完成各個欄位的填寫後，可以使用底部的匯出與佐證資料上傳卡：

📷 佐證資料上傳與圖說

您可多張上傳相關照片，並為每張照片填寫簡短圖說（圖說非常關鍵，務必寫明照片中的活動主題與你所在的具體位置）。所有上傳的圖片均會轉為本機編碼，保存於您當前的瀏覽器中，並能完美的整合輸出到 PDF。

🔗 合併本檔內容

快速將這份學習歷程下所有的填寫欄位，依格式組裝成一篇結構分明、具易讀性的排版文章，供您預覽或直接複製全文。

📝 下載 Word

將整篇學習歷程打包下載為 Word 格式的 `.doc` 文件，供您在 Microsoft Word 中進一步精修、調整格式或合併至其他大型文件中。

🖨️ 列印／下載 PDF

觸發列印友善版面。在彈出的列印視窗中，將目的地設為「另存為 PDF」，即可輸出包含主題文字、摘要、完整段落，以及佐證圖片的排版學習歷程成果檔案！
